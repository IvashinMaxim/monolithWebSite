# AutoSite

## Описание проекта
Платформа для размещения и управления объявлениями о продаже автомобилей. Пользователи могут регистрироваться, авторизовываться и управлять своими объявлениями. Проект демонстрирует навыки работы с Java, Spring Boot и базами данных.

## Функционал
- Регистрация и авторизация пользователей
- Управление собственными объявлениями (CRUD)
- Поиск и фильтрация объявлений по параметрам автомобиля

## Технологии
- **Backend:** Java 8+, Spring Boot, Spring Data JPA, Hibernate
- **База данных:** PostgreSQL
- **Безопасность:** Spring Security
- **Frontend:** Thymeleaf

## Особенности проекта / Чему научился
- Проектирование сущностей и связей в базе данных с большим объёмом данных
- Реализация регистрации и авторизации пользователей
- Интеграция backend с frontend (Thymeleaf)
- Работа с CRUD операциями и фильтрацией данных
- Использование паттернов проектирования и архитектурных принципов Spring
